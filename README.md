# Pewny Leasing

Platfoma front-endowa aplikacji Kalkulatora Ofertowego "Pewny Leasing" oraz wizerunkowej strony Landing Page, pozyskująca leady.

## Uruchamianie LokaLnie
```bash
npm install
npm run dev
```

## Technologia
- React 18, Vite + TypeScript, SWC
- Tailwind CSS (Zmienne Kolorystyczne: zółty z makiety, ciemny motyw, stylizacja glassmorphism)
- Dostępność (WCAG 2.0 / aria-labels na UI / outline:none fallback na elementach)
- Hook-based i18n (Tylko PL/EN wsparcie docelowe)

Projekt korzysta stricte ze zoptymalizowanych małych wielorazowych komponentów oraz unikamy zewnętrznych, ciężkich bibliotek (brak Component Libraries), co gwarantuje natychmiastowe ładowanie UI.
